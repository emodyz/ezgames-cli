# Usage
<!-- usage -->
```sh-session
$ npm install -g ezgames-cli
$ ezgames COMMAND
running command...
$ ezgames (-v|--version|version)
ezgames-cli/0.0.0 darwin-x64 node-v15.11.0
$ ezgames --help [COMMAND]
USAGE
  $ ezgames COMMAND
...
```

## `ezgames help [COMMAND]`

display help for ezgames

```
USAGE
  $ ezgames help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
